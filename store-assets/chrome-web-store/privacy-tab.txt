==================================================
SINGLE PURPOSE  (one text box)
==================================================
Compliance Central lets Michigan automotive dealers screen buyers and co-buyers for OFAC sanctions, MDOS repeat-offender status, and vehicle title/lien status from a single browser side panel, and produce a printable record of the result.


==================================================
PERMISSION JUSTIFICATIONS  (one box per permission)
==================================================

sidePanel —
The entire user interface is a Chrome side panel; this permission opens and renders it.

storage —
Saves the dealer's compliance history, preferences, and (optionally) a custom backend API key locally on the device.

unlimitedStorage —
The OFAC SDN sanctions dataset plus historical compliance records (with evidence screenshots) can exceed the default storage quota.

alarms —
Schedules an automatic daily refresh of the local OFAC sanctions list so screenings use current data.

Host permission https://data.opensanctions.org/ —
Downloads the public OFAC SDN sanctions list used for on-device screening.

Host permission https://compliance-central-api.fly.dev/ —
The dealer's backend endpoint that performs the Repeat Offender and Title/Lien checks against the Michigan Department of State portal. It also relays the optional phone license-scan: the scanned fields are end-to-end encrypted on the phone, so the backend only passes along an encrypted, single-use package it cannot read (deleted on pickup; the license image is never transmitted).

Are you using remote code?  →  NO
(All code is bundled in the package; nothing is fetched and executed remotely.)
